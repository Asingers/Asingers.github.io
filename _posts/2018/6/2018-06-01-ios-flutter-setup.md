### 下载

	git clone -b beta https://github.com/flutter/flutter.git
	通常我们安装在用户根目录下
	
### 配置环境变量

	export PATH=your src/bin:$PATH
	
### 安装

	flutter doctor
	

- ps:如果遇到一些model没有安装，比如six? 我是这么做的


		brew reinstall python@2
		pip install six
	
### 创建
	
	flutter creat simple
	
### 运行

	flutter run
